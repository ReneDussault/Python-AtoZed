## Code Analysis Questions (10 points)

### 6. What will this code output?
```python
class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        return f"{self.name} says Woof!"

my_dog = Dog("Buddy")
print(my_dog.bark())
```
a) Dog says Woof!
b) Buddy says Woof!
c) name says Woof!
d) Error

### 7. What will this code output?
```python
class Animal:
    def speak(self):
        return "Some sound"

class Cat(Animal):
    def speak(self):
        return "Meow"

my_cat = Cat()
print(my_cat.speak())
```
a) Some sound
b) Meow
c) Both  
d) Error

### 8. What will this code output?
```python
class Counter:
    def __init__(self):
        self.count = 0
    
    def increment(self):
        self.count += 1

c1 = Counter()
c2 = Counter()
c1.increment()
print(c1.count, c2.count)
```
a) 0, 0  
b) 1, 1  
c) 1, 0  
d) 0, 1
